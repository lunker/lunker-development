# Shell Script  
---



### 변수

- $변수명 으로 사용한다.  
- 값을 대입할 때에는 "$"없이 그냥 사용한다.  
- 대입연산을 수행할 때에는 공백이 존재하면 안된다.  
-


### Argument  

- $1 ~ $n : 넘겨진 인자들
